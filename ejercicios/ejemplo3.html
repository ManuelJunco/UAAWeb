La Casa de El

Problema detectado

Información del problema

Mal uso de atributo alt en la imagen. Alt debe usarse como texto alternativo a la imagen. Las razones varían desde que el navegador no pueda cargar la imagen hasta que el usuario no pueda ver la imagen por causas como ceguera total o ceguera de colores.

En este caso, se utiliza como atributo alt el nombre que se le ha dado a la propia imagen: una serie de caracteres sin significado alguno

Solución propuesta

Incluir un atributo alt coherente, como se tiene para el atributo title.
